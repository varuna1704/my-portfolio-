⚡ THE FASTEST SETUP (5 MINUTES):

Open Terminal/Command Prompt and paste:

    npm create vite@latest my-portfolio -- --template react
    cd my-portfolio
    npm install
    mkdir -p public/images src/components
    npm run dev

Then:
  1. Copy profile.png → public/images/profile.png
  2. Copy portfolio-with-image.jsx → src/components/Portfolio.jsx
  3. Update src/App.jsx (instructions in QUICK_START_CHEATSHEET.md)
  4. Open http://localhost:5173/ in browser

Done! 🎉
