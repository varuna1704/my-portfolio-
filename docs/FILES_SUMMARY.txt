=============================================================================
                        TROUBLESHOOTING
=============================================================================

Image not showing?
→ Check: public/images/profile.png exists
→ Filename must be exactly: profile.png
→ Try hard refresh: Ctrl+Shift+R

Port already in use?
→ Run: npm run dev -- --port 3000

Component not found?
→ Check: src/components/Portfolio.jsx exists
→ Verify: App.jsx imports it correctly

Styling looks broken?
→ Clear browser cache: Ctrl+Shift+R
→ Restart dev server: Close terminal, run npm run dev again

Still stuck?
→ Read: STEP_BY_STEP_SETUP.md (Troubleshooting section)
→ Check: All files in correct locations
→ Delete: node_modules folder, run: npm install again
